    回答のコツ
    • 出力値が分かっている時は、逆算して考える。
    • 「整数のx乗の一の位」は、一の位のx乗の値の一の位になる。
      (ex.19の3乗の一の位は、9の3乗=9×9×9=81×9=729より、9)
    • 「偶数+奇数＝奇数」、「偶数+偶数＝偶数」、「奇数+奇数＝偶数」

    基礎編

    「10,11」のように2つの連続する2ケタの整数を、それぞれに2乗して足し合わせた数のうち、 一の位が3となるものはいくつあるか。

    1. 12
    2. 14
    3. 16
    4. 18
    5. 20
    正答：4

    0~9までの数とそれを二乗したものの一覧は下図のようになる。


    上図より、2つの連続する整数の二乗の和の一の位が3になるものは、
    ”2と3”の組み合わせと、”7と8”の組み合わせの2種類である。
    (4+9=13 と 49+64=113)

    この2種類の組み合わせが2ケタの整数となるのは、
    12,13 ~ 92,93 の9つと、
    17,18 ~ 97,98 の9つの、計18個

    よって正解は4の18個。

    応用編

    2ケタの整数A,B,C,D(A＜B＜C＜D)がある。この4つの数字のうち偶数が1個で、奇数が3個であった。 このうち2つの数字を足すと全部で6通りの組み合わせがあり、そのうち小さいほうから4つは43,46,50,55である。
    このとき、Dの一の位と十の位の和はいくつか。

    1. 6
    2. 7
    3. 8
    4. 9
    5. 10
    正答：1

    和の組み合わせ6つというのは、A+B,A+C,A+D,B+C,B+D,C+D の6つである。

    A＜B＜C＜Dより、一番小さい和はA+Bで、二番目に小さい和はA+Cである。
    したがって、A+B=43、A+C=46とわかる。

    ここで、「偶数+奇数＝奇数」かつ「奇数+奇数＝偶数」より、
    A+B=43だから、AかBのいずれかが奇数であることが推測でき、
    A+C=46より、AとCが偶数であることも推測できる。
    よって、Bが偶数で、A,C,Dの3つが奇数であるとわかる。
    (∵問題文より、A,B,C,Dのうち1つだけが偶数のため。)

    三番目に小さい和はA+DかB+Cのいずれかであると推測できるが、
    Bが偶数であることが分かっているため、「奇数+奇数」となるA+Dが三番目に小さい和と推測できる。
    よって、A+D=50。 またそれにより、四番目に小さい和はB+Cであることが分かるため、B+C=55である。

    今までの過程で、
    A+B=43、A+C=46、A+D=50、B+C=55ということが分かっている。 よってこの四式を連立すると下図のようになる。

    上記の計算より、D=33だから、3+3=6。

    よって正解は1。

    頻出テーマ2.約数・倍数

    回答のコツ
    • 整数条件のときに分数が出てきたら、「倍数」が使えないか考える。
      (ex.x=2/3y で「xが整数」なら、「yは3の倍数」かつ「xは2の倍数」)
    • 「〇〇で割り切れる」や「〇〇ごと」などの記述があったら、「倍数」が使えないか考える。
    • 「pで割ってq余る数x」は「x=pm+q(mは1以上の整数)」で表せる。
      (ex..8で割って2余る数xは、x=8m+2となる。
      　⇒　m=1ならx=10 → 10÷8=1余り2
      　　　m=2ならx=18 → 18÷8=2余り2)

    基礎編

    A~Dの4人が、100点満点の試験を受けた。4人の得点について、次のことが分かっているとき、 Aの得点とBの得点を足し合わせた得点はどれか。ただし、試験の得点は全て整数とし、 0点の者はいないものとする。


    1. 36点
    2. 60点
    3. 96点
    4. 120点
    5. 144点
    正答：4

    A~Dの得点をa~dとおく。

    「Aの得点は、Bの得点の5/7倍」より、a=5/7bであるから、
    bが7の倍数かつ、aが5の倍数であることが分かる。

    「Bの得点は、Cの得点の5/3倍」より、b=5/3cであるから、 cが3の倍数かつ、bが5の倍数であることが分かる。

    bが7の倍数かつ5の倍数であるから、5と7の最小公倍数である35の倍数であることが推測できる。
    試験は最高で100点のため、bは35か70であることが分かる。

    (ⅰ)b=35のとき、
    a=5/7×35=25、c=3/5b=3/5×35=21

    (ⅱ)b=70のとき、
    a=5/7×70=50、C=3/5b=3/5×70=42

    ここで、「Cの得点は、Dの得点の2倍」より、c=2dであるから、 Cが2の倍数(偶数)であることが分かる。 よって、c=21でなく、c=42であることが確定する。

    よって、a=50、b=70より
    a+b=120となる。

    よって正解は4。

    応用編

    1~100までの数が1つずつ書かれた100枚のカードがある。この中から、ある2ケタの整数xで割り切れる数の書かれたカードを 取り除いたところ、取り除かれたカードは4枚あった。さらに、残ったカードの中から8で割り切れるカードを取り除いたところ、 新たに取り除かれたカードは11枚であった。この時、2ケタの整数xの十の位と一の位の数の和として正しいものは、次のうちどれか。

    1. 3
    2. 4
    3. 5
    4. 6
    5. 7
    正答：2

    「整数xで割り切れる数の書かれたカードを取り除いたところ、取り除かれたカードは4枚あった。」 とあるため、1~100のなかでxの倍数が4つあるということである。
    よって、xの候補として考えられるのは「21,22,23,24,25」の5つ。

    次に、「残ったカードの中から8で割り切れるカードを取り除いたところ、新たに取り除かれたカードは11枚」 とあるが、100÷8=12.5より、本来であれば1~100のなかで8の倍数は12個あったため、
    最初に取り除いたカードのうち、8の倍数が1つだけ入っているものを探す。

    (ⅰ)最初に取り除いたカードが21のとき、
    取り除かれたカードは「21、42、63、84」で、このうち8の倍数は0個のため✕。 (ⅱ)最初に取り除いたカードが22のとき、
    取り除かれたカードは「22、44、66、88」で、このうち8の倍数は1個のため〇。 (ⅲ)最初に取り除いたカードが23のとき、
    取り除かれたカードは「23、46、69、92」で、このうち8の倍数は0個のため✕。 (ⅳ)最初に取り除いたカードが24のとき、
    取り除かれたカードは「24、48、72、96」で、このうち8の倍数は4個のため✕。 (ⅴ)最初に取り除いたカードが25のとき、
    取り除かれたカードは「25、50、75、100」で、このうち8の倍数は0個のため✕。

    (ⅰ)~(ⅴ)より、x=22であるから、2+2=4。

    よって正解は2。

    頻出テーマ3.数量問題

    回答のコツ
    • その条件のみでは確定せず、複数の条件を組み合わせることで確定することが多いため、 考えられるパターンをすべて把握して適した答えを素早く見つけられるようにしておく。
    • ほかのテーマに比べ文章が長くなることが多いため、数多くの問題をこなすことで読解力も身につけておきたい。
    • 図や表を作ると解きやすくなることがあるため、頭の中だけで考えず、図や表を積極的に用いて試行錯誤しながら解いていく。

    基礎編

    ある職場では、表のような消耗品を3回に分けて必要個数だけ購入した。

    〇1回目は、クリアファイルを除く3種類の消耗品をそれぞれ1個以上購入し、合計金額は1200円であった。
    〇2回目および3回目は、ともに4種類全ての消耗品をそれぞれ1個以上購入し、合計金額は、2回目が2300円、 3回目が1500円であった。

    このとき、確実にいえるのはどれか。


    1. 1回目に消しゴムを2個購入した。
    2. 1回目にガムテープを3個購入した。
    3. 2回目に付せん紙を1個購入した。
    4. 2回目にガムテープを3個購入した。
    5. 3回目に消しゴムを1個購入した。
    正答：3

    1回目について、
    「クリアファイルを除く3種類の消耗品をそれぞれ1個以上購入した」ため、とりあえず1個ずつ購入した金額を調べると、 110+170+290=570円。
    そして「1回目の合計金額は1200円だった」ため、残金は1200-570=630円
    630円で購入できるのは、付せん紙2個とガムテープ1個で170×2+290=630円となる。

    1回目終了時点での残りの消耗品の必要個数は、
    消しゴムが6個、付せん紙が2個、ガムテープが6個、クリアファイルが2個。

    次は、合計金額が少ない3回目について検討する。
    「3回目は、ともに4種類全ての消耗品をそれぞれ1個以上購入した」ため、とりあえず1個ずつ購入した金額を調べると、 110+170+290+530=1100円。
    そして「3回目の合計金額は1500円だった」ため、残金は1500-1100=400円
    400円で購入できるのは、消しゴム1個とガムテープ1個で110+290=400円となる。

    1回目と3回目終了時点での残りの消耗品の必要個数は、
    消しゴムが4個、付せん紙が1個、ガムテープが4個、クリアファイルが1個。

    残りの必要個数で合計金額を調べると
    110×4+170+290×4+530=440+170+1160+530=2300円となるため、2回目の合計金額と一致する。

    よって、各回で購入した消耗品の個数は
    1回目...消しゴムが1個、付せん紙が3個、ガムテープが2個、クリアファイルが0個
    2回目...消しゴムが4個、付せん紙が1個、ガムテープが4個、クリアファイルが1個
    3回目...消しゴムが2個、付せん紙が1個、ガムテープが2個、クリアファイルが1個

    よって正解は3。

    応用編

    大、中、小の3つのサイズの莢があり、大サイズの莢には豆が5粒、小サイズの莢には豆が3粒入っている。 また、中サイズの莢には豆が4粒または5粒入っているが、その数は莢を開いてみなければわからない。
    今、A~Dの4人がそれぞれいくつか莢を取り、その莢から豆を取り出して、自分の年齢の数だけ豆を集めることとした。 各人が次のように述べているとき、4人の年齢の合計はいくつか。
    A:「大サイズの莢を2個、中サイズの莢を2個、小サイズの莢を2個取ったところ、自分の年齢と同じ数の豆が入っていた。」
    B:「中サイズの莢を4個取ったところ、自分の年齢より4粒多く豆が入っていた。また、4個の莢のうち少なくとも1個には、豆が5粒入っていた。」
    C:「自分の年齢はAとBの年齢の合計と同じである。1個だけ小サイズの莢を取り、残りは大サイズの莢を取ったところ、 自分の年齢と同じ数の豆が入っていた。」
    D:「私は、Bより2歳年上である。小サイズの莢を5個以上取ったところ、自分の年齢と同じ数の豆が入っていた。」

    1. 91
    2. 92
    3. 93
    4. 94
    5. 95
    正答：1

    まず、Aの年齢について考える。
    Aは「大サイズの莢を2個、中サイズの莢を2個、小サイズの莢を2個取ったところ、自分の年齢と同じ数の豆が入っていた。」と述べており、
    中サイズは「4，4」、「4，5」、「5，5」の3パターンある。
    よってAの年齢は、24歳、25歳、26歳のいずれかである。

    次にBの年齢について考える。
    Bは「中サイズの莢を4個取ったところ、自分の年齢より4粒多く豆が入っていた。また、4個の莢のうち少なくとも1個には、豆が5粒入っていた。」と述べており、
    中サイズ「4，4，4，5」、「4，4，5，5」、「4，5，5，5」、「5，5，5，5」の4パターンで、とった豆の数は17個，18個，19個，20個のいずれか。
    よって、Bの年齢は、13歳、14歳、15歳、16歳のいずれか。

    次にCの年齢について考える。
    Cは「自分の年齢はAとBの年齢の合計と同じである。1個だけ小サイズの莢を取り、残りは大サイズの莢を取ったところ、自分の年齢と同じ数の豆が入っていた。」と述べているため、
    Cの年齢は、8歳、13歳、18歳、23歳、28歳、33歳、38歳、43歳....というような様子になる。
    しかし、Cの年齢は、AとBの年齢の合計のため、37歳~42歳の間になるから、38歳しかありえない。

    次にDの年齢について考える。
    D:「私は、Bより2歳年上である。小サイズの莢を5個以上取ったところ、自分の年齢と同じ数の豆が入っていた。」 と述べているため、 Dの年齢は、15歳、18歳、20歳...というような様子になるが、
    Bの年齢が13歳、14歳、15歳、16歳のいずれかのため、Dは15歳か18歳となり、さらにBは13歳か16歳となる。

    (ⅰ)Bの年齢が13歳のとき、
    Cの年齢が38歳であることから、Aの年齢が25歳とわかる。

    (ⅱ)Bの年齢が16歳のとき、
    Cの年齢が38歳であることから、Aの年齢が22歳とわかる。
    しかし、Aは24歳~26歳の間のため不適。

    よって、(ⅰ)の時のみがすべての条件を満たすとわかる。
    よって、4人の年齢の合計は、25+13+38+15=91。

    よって正解は1。

    頻出テーマ4.速さ・距離・時間

    回答のコツ
    • 「何を基準にして式を作るか」や「何を未知数にとるか」を考える。 (問題を多くこなして慣れることが重要)
    • 「時速」や「30分」などがあったら単位をそろえる。
    • 速さ・距離・時間について、比を使うと解きやすくなることがあるため意識する。

    基礎編

    地点Aから地点Bまでが上り坂、地点Bから地点Cまでが下り坂の一本道がある。 地点Aを自転車で出発し、地点Cで15分間の休憩後、折り返し、復路の地点Bで8分間の休憩後、 地点Aに戻ったところ、1時間15分かかった。 地点Aから地点Cまでの距離はどれか。ただし、上り坂は時速6km,下り坂は時速20kmで走行する。

    1. 3250m
    2. 3500m
    3. 3750m
    4. 4000m
    5. 4250m
    正答：4

    状況としては、上図のようになっている。
    上図を参考に、地点Aから地点Bまでをx kmとおき、地点Bから地点Cまでをy kmとおく。 求めるものは地点Ａから地点Ｃまでの距離のため、(x +y)である。

    「往路にかかった時間」 +「復路にかかった時間」 + 「休憩時間」 = 「全部にかかった時間」...(⁎)という式を作っていく。

    まず、「往路にかかった時間」について考えていく。
    往路では、地点Aから地点Bまでが上り坂になるため時速6km/h、
    復路では、地点Bから地点Cまでが下り坂になるため時速20km/hで走行したため、
    (x/6 + y/20 )分　となる。

    次に、「復路にかかった時間」について考えていく。
    往路では、地点Cから地点Bまでが上り坂になるため時速6km/h、
    復路では、地点Bから地点Aまでが下り坂になるため時速20km/hで走行したため、
    (y/6 + x/20 )分　となる。

    「休憩時間」は8分+15分=23分だが、速度が時速のため、(23/60)分となり、
    「全部にかかった時間」は1時間15分のため(75/60)分となる。

    よって、(⁎)式は
    (x/6 + y/20 ) + (y/6 + x/20 ) + 23/60 = 75/60 で両辺を60倍して
    10x + 3y + 10y + 3x + 23 = 75
    13x + 13y = 52
    x + y = 4

    よって、地点Aから地点Cまでの距離は4km = 4000mである。 よって正解は4。

    応用編

    A~Cの3人が、X町からY町へ同じ道を通って行くことになった。まずAが 徒歩で出発し、次に30分遅れてBがランニングで出発し、最後にCがB より一時間遅れて自転車で出発した。その結果、Cが、出発後30分で Aを追い越し、さらにその30分後にBを追い越したとき、AとCとの距離が 6kmであったとすると、Bの速さはどれか。ただし、3人の進む速さはそ れぞれ一定とする。

    1. 時速7km
    2. 時速8km
    3. 時速9km
    4. 時速10km
    5. 時速11km
    正答：2

    A,B,Cの速さをそれぞれa km/h ,b km/h ,c km/hとおく。

    まず、a と c の関係について考えていく。
    BはAより30分遅れて出発し、そのBよりCは1時間遅れて出発したため、 CはAより1時間30分遅れて出発したことが分かる。
    そして、Cが、出発後30分でAを追い越したため、 Aが2時間かかったところをCが30分で進んだことが分かる。
    よって、a : c = 30 : 120 となり、c = 4a となる。

    次に、まず、b と c の関係について考えていく。
    BよりCは1時間遅れて出発し、その60分後にCがBを追い越したため、 Bが2時間かかったところをCが1時間で進んだことが分かる。
    よって、b : c = 60: 120 となり、c = 2b となる。

    CがBを追い越したとき、AとCとの距離が6kmであったとあり、 このとき、Aはスタートしてから2時間30分たち、Cはスタートしてから1時間たっている。
    よって、60c - 150a = 6 となり、これにc = 4a を代入して計算すると a = 1/15 となり、 c = 4/15 を得る。これをc = 2b に代入すると b = 8 となる。

    よって正解は2。
